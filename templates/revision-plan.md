# Revision Plan Template

Use for `revise`, `humanize`, and `finish` modes.

## Revision target

Pick or infer one primary target:
- clearer
- shorter
- warmer
- sharper
- more direct
- more persuasive
- more personal
- more natural
- less corporate
- less AI-ish
- more literary
- more executive
- more emotionally restrained
- more memorable

If targets conflict, prioritize the stated audience and purpose.

## Output shape

```markdown
## Revision focus
[What changed and why.]

## Revised version
...

## Change notes
- ...
```

For deeper revisions:

```markdown
## Weak spans
| Original | Issue | Replacement |
|---|---|---|

## Revised version
...

## Remaining choices
- ...
```

## Rules

- Preserve meaning unless asked to rethink substance.
- Do not erase useful personality.
- Do not invent details.
- Prefer surgical edits when the draft is structurally sound.
- Rewrite globally only when structure, audience fit, or argument is broken.
